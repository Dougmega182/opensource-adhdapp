# Assets Directory

Place your app icons and splash screens here.

## Required Assets (for Expo)

- **icon.png** - App icon (1024x1024)
- **splash.png** - Splash screen (1284x2778 recommended)
- **adaptive-icon.png** - Android adaptive icon (1024x1024)
- **favicon.png** - Web favicon (48x48)

## Placeholder Assets

The app will run without these files, but you should add proper assets before publishing.

### Quick Asset Generation

You can use online tools to generate these:
- https://www.appicon.co/
- https://icon.kitchen/
- https://makeappicon.com/

Or use the Expo asset generator:
```bash
npx expo optimize
```
