# ADHD Assistant

Open-source ADHD productivity app with AI-powered task breakdown, brain dump processing, and Pomodoro timers.  
Designed to help users manage tasks, focus on work sessions, and simulate ADHD-friendly productivity workflows.

---

## Features

- **AI Task Breakdown** – Automatically breaks down high-level tasks into smaller actionable subtasks.  
- **Brain Dump Processor** – Convert free-form notes into structured tasks saved in SQLite.  
- **Pomodoro Timer** – CLI-based work sessions to boost focus and track time.  
- **Workspace Generator** – Creates base app structure for React Native MVP.  
- **SQLite Storage** – Persistent task and brain dump storage.  

---

## Requirements

- Python 3.11+  
- Windows / macOS / Linux  
- Optional: Ollama AI (stubbed in this version)
